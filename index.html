Tecnologias FullStack

#1

Java Script

Linguagem de Programação
que permite implementar
itens complexos em páginas web

#2

React.JS

Biblioteca JavaScript para
criar interfaces de usuário

#3

Yarn

Gerenciador de pacotes
para aplicar comandos
prontos ao código de uma aplicação.

#4

Node.JS

Node.js é um runtime
JavaScript construído no
mecanismo V8 do Chrome.

#5

Git

Git é um sistema de controle
de versão distribuído
gratuito e de código aberto.

#6

MongoDB

Banco de dados que
armazena dados em
documentos do tipo JSON.
